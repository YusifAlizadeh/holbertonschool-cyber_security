# 🛡️ Holberton Cybersecurity

This repository contains my cybersecurity learning projects from the Holberton cybersecurity program.
Here I practice fundamental security concepts, tools, and techniques used in real-world environments.

The goal of this repository is to document my progress while building practical cybersecurity skills.

---

## 📚 Repository Structure

### 🌐 Web Application Security

Focus on identifying and understanding vulnerabilities in web applications.

Topics may include:

* HTTP / HTTPS fundamentals
* Web vulnerabilities (XSS, SQL Injection, CSRF)
* Authentication & session security
* Basic web exploitation techniques

Directory: `web_application_security/`

---

### 🖥️ Linux Security

Security practices and concepts related to Linux systems.

Topics may include:

* Linux permissions and user management
* Process monitoring
* System hardening basics
* Secure configuration practices

Directory: `linux_security/`

---

### 🌐 Network Security

Core networking concepts and how they relate to security.

Topics may include:

* Packet analysis
* Understanding common network attacks

Directory: `network_security/`

### 🔐 Cybersecurity Basics

Foundational cybersecurity concepts and principles.
